# Weather API WhatsApp

## Overview
This n8n workflow fetches weather data and sends it to WhatsApp using Twilio.

## Technologies
- n8n
- Weather API
- Twilio
- HTTP Request

## Features
- Fetches live weather
- Sends WhatsApp messages automatically
- Easy to customize

## Setup
1. Import the workflow.
2. Add your API keys.
3. Configure Twilio.
4. Run the workflow.
